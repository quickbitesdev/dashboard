QuickMailBites

PRE-LAUNCH

Overview

Native cross-platform desktop email client built with Flutter. The only email client that reads raw MIME emails directly from AWS S3 buckets. Supports IMAP, S3, and Gmail OAuth.

Features

  • S3 email reading for AWS SES users
  • Native desktop app (not Electron)
  • IMAP + S3 + Gmail OAuth support
  • Keyboard-first navigation (j/k, r, f, c)
  • Dark glassmorphism theme
  • System tray with unread badge
  • Auto-update with semver versioning
  • Draggable resizable panels
