Built for developers who use AWS

The only email client that natively reads S3 buckets. No workarounds, no forwarding.

S3

S3 Email Reading

Read raw MIME emails directly from your AWS S3 bucket. Perfect for SES users who route inbound email to S3.

N

Actually Native

Built with Flutter. Not Electron, not a web wrapper. Real native performance with low memory usage and instant startup.

3x

IMAP + S3 + Gmail

One client for all your email. Standard IMAP, AWS S3 buckets, and Gmail OAuth all work together.

kb

Keyboard-First

Navigate with j/k, reply with r, forward with f, compose with c. Built for people who hate reaching for the mouse.

UI

Dark Glassmorphism

Modern dark theme with blur effects and green accents. Draggable, resizable panels that adapt to your workflow.

up

Auto-Update

Self-updating binary with semver versioning. System tray icon with unread badge keeps you informed.
