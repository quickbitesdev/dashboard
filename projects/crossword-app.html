Architecture

Browser → fetches puzzle JSON from crossword-data GitHub Pages
crossword-data ← pushed daily by generate_crossword.py
generator → calls Ollama (qwen3:8b) for word/clue generation
Flutter app → same JSON API, same puzzle data (future)
